# Online Examination System (Django)

## Project Description
This is a web-based Online Examination System built using Django.
Students can log in, attend multiple-choice exams, and view results.
Admins can manage students and questions using Django Admin Panel.

## Technologies Used
- Python 3
- Django
- HTML
- CSS
- SQLite (default Django database)

## Features
- Student Login
- MCQ Based Exam
- Automatic Evaluation
- Result Calculation (Score & Percentage)
- Admin Panel for Managing Questions & Students
- Session-based Authentication
